# BlinkWise AI

BlinkWise AI is a powerful health-tracking application that uses computer vision to monitor eye health and provide insights into digital strain and blink patterns.

## Features
- Real-time blink detection using MediaPipe.
- Personalized health dashboards.
- Session tracking and history.
- AI-driven insights for better eye health.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion.
- **Backend**: Node.js, Express.
- **AI/ML**: MediaPipe Face Landmarker.

## Getting Started
1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Start the backend: `cd server && npm start`
